# Meridian Introduction

Meridian is an Engineering Document Management System (EDMS). It helps you to manage technical documents and drawings through their lifecicle. You can store, organize, and control access to your documents in one place. You can also manage document revisions and workflows, ensuring seemless collaboration.

Meridian helps you meet industry standards and regulatory requirements. It keeps a single, reliable source of truth for all engineering documents. 

## Meridian Enterprise Server
Meridian Enterprise Server is an on-premises product suit that includes:
- **PowerWeb**: Create and manage engineering data.
- **Explorer**: Search and view documents and drawings.
- **Configurator**: Configure the system workflow, document types, and more.
- **Administrator**: Manage users, licenses, and vaults.
- **Administrator Console**: Manage users, licenses, renditions, repositories, and more.

## Meridian Cloud
Meridian Cloud is a cloud-based product suite managed by Accruent. We host it on Microsoft Azure data centers. It includes:
- **Power**: Create and manage engineering data.
- **Cloud and Classic Explorer**: Search and view documents and drawings.
- **Portal**: Collaborate with external users on engineering projects.
- **APIs**: Integrate Meridian with CMMS, GIS, ERP, and EAM systems. With the API, you can create and update assets in Meridian. You can also establish relationships between assets and documents.

## Meridian Mobile
Meridian Mobile is an app for iOS, Android, and Windows. It lets you access asset data while you are on the move.
