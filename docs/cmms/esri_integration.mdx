# Esri ArcGIS Integration

Connect Maintenance Connection assets to Esri's mapping platform. The integration displays your organization’s Esri map in an embedded view in the Asset map, so you can interact with assets and location data directly within the application. With this integration, you can easily locate equipment, plan maintenance activities, and analyze patterns across your infrastructure.

## Prerequisites

Your system administrator must configure the Esri ArcGIS integration before you can use this feature.

## Link assets with Esri objects

Connect your Maintenance connection assets with Esri objects through the Global GIS ID in three different ways to speed up your workflow.

### Link through the Asset module

1. Go to the **Assets** module.
2. Select the asset you want to link.
3. In the **Location info** section, select **Sync asset with GIS**.
4. Enter the **Global GIS ID**.
5. Click **Link**.

### Link through the Unlinked tab in the Asset map

1. Open the **Asset map**.
2. Select the **Unlinked assets** tab.
3. Select **Link to GIS feature**.
4. Choose an existing Esri object in the map view.
5. In the dialog, select **Link to asset**.
6. In the side panel, select **Confirm**.

The system links the MC asset to the Esri object.

### Link Esri object to a new MC asset in the map view

1. Open the **Asset map**.
2. In the map view, select an unlinked asset.
3. Select **Link to asset**.
4. Select **Create link to new asset**.
5. Enter the asset details.
6. Select **Create**.

The Esri object links to a newly created MC asset. To add more details, open the asset in the **Assets** module.

## Remove link with Esri objects

Delete the Global GIS ID to remove the link with the Esri object.

1. Go to the asset you want to remove the link.
2. In **Location info > Global GIS ID**, select **Remove link with GIS**.
3. Select **Delete**.
