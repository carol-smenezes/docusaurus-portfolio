# Asset Map

Use the Asset map to manage assets, work orders, and maintenance workflows through an interactive geospatial interface directly in Maintenance Connection. The map helps you visualize locations, understand asset relationships, and quickly access related records.

You can open the map view in the following ways:

* From the toolbar, select **Asset map**.
* In the **Location info** section of an asset or work order, select **View on asset map**.

## Prerequisites

* You can access the **Asset map** once your system administrator configures the Esri integration and grants the necessary access rights.
* To view an asset on the map, the asset must include location information, such as latitude and longitude coordinates.
* If the asset isn't linked to the GIS feature, the asset displays in the **Unlinked** tab.

## Map view

The map view includes tools that help you navigate, orient, and explore locations. It displays your organization's Esri map in an embedded view, so you can interact with assets and location data directly within the application.

The available navigation and display tools depend on your organization's Esri widget configuration and can include:

* **Zoom in:** Select `+`, press the `+` key, or scroll the mouse wheel up.
* **Zoom out:** Select `-`, press the `-` key, or scroll the mouse wheel down.
* **Default map view:** Return to the predefined zoom level and location.
* **Find my location:** Center the map on your current location.
* **Reset map orientation:** Reset the map to the north-facing orientation.
* **Change map orientation:** Right-click and drag to rotate the map view.
* **Enter full screen:** Expand the map to full-screen mode for easier viewing.
* **Search:** Find an address or place by entering a search term.
* **Basemap gallery:** Change the underlying map style, such as street and imagery views.
* **Layer list:** Show or hide map layers.

## View assets and work orders

In the **Assets** and **Work orders** tabs of the map view, you can:

* Apply filters
* Sort results
* View assets or work orders details. Selecting a record on the map highlights its location and provides quick access to related information.

### Apply filters

1. In the **Asset map**, select **Filter**.
2. In the filter pane, enter or select one or more filters.
> Clear **Include assets outside of view** to display results strictly limited to assets located within the current map bounds.
3. Select **Apply**.

The filter icon indicates the number of applied filters. 

To remove the applied filters, click **Filter > Clear all**, then click **Apply**.

### Sort list

Sort assets and work orders in ascending or descending order.

* In the **Assets** tab,  sort by **Asset ID** or **Name**.
* In the **Work orders** tab, sort by **WO#** or **Target date**.

Select the arrow position to indicate the sort order:

* Up arrow for ascending order.
* Down arrow for descending order.

### View unlinked assets

Find assets that have latitude and longitude coordinates but are not linked to the GIS feature in the **Unlinked assets** tab.
