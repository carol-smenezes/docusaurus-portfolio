Task-Oriented Samples

📄️Sign up

Start using Meridian Cloud by accepting the Meridian Cloud invitation email. Once the system administrator invites you to the Meridian Cloud tenant, you receive an email with a link to accept the invitation and enter your identity information. If you don't receive the invitation email, check with your system administrator to confirm the email address to which the invitation was sent.

📄️Sign in

If you have a user account, sign in and start using Meridian Cloud. If you don't have an account yet, contact your system administrator to request access and sign up.

📄️View Map in Mobile

Use the map in MC Kinetic to find assets and work orders, and to identify where technicians need to go.

📄️Configure Esri ArcGIS Integration

Configure the ArcGIS integration to connect your Maintenance Connection assets and work orders directly to Esri’s mapping platform.

📄️Pin Photos in Mobile

Pin photos directly to drawing artefacts for ongoing use using the mobile app for Android and iOS. You can pin photos to a drawing as visual supporting information regardless of its workflow state or version, similar to adding a comment. For example, field technicians can pin photos to provide real-world context, making artefacts easier to understand.
